Bluetooth Transfer User Guide


The application requires Linux, Python Gtk and Bluez5 to run.

Terminate or uninstall other bluetooth application before using this application. After starting, application is ready to receive files and to handle pairing. Pairing is required and pairing is needed only once. Bluez5 stores pairing information for later use.

Downloaded files go to directory: ~/.cache/obexd

Last downloaded file name is shown in the application. From the phone you can see when the transfer is finished.

To send files to the phone, set bluetooth on in your phone, click Send Files, then select device and OK and finally select files and click OK. You can click Cancel to cancel the operation. Last uploaded file name is show in the application.

To restart bluetooth drivers and this application, click Restart BT and give your sudo (or root, if sudo not working) password.

To close the application, click the window close button (X).
